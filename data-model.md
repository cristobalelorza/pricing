# Precio - Data Model

## Source file
`app/models.py`

## Models

### DealInput
User-provided deal information.

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| service_description | str | yes | What service is being offered |
| business_url | str | yes | URL of client's business website |
| client_context | str | no (auto-populated) | Auto-generated by Research Agent from business_url |
| constraints | str | no | Timeline, budget caps, delivery constraints |
| budget_hint | float | no | Optional budget signal from client |
| currency | Currency | yes (default USD) | USD, EUR, CLP, GBP |

### AgentProposal
Output from Cost, Market, Value specialist agents.

| Field | Type | Description |
|-------|------|-------------|
| agent_name | str | Which agent produced this |
| price_floor | float | Minimum viable price |
| price_target | float | Recommended price |
| price_stretch | float | Aspirational price |
| reasoning | str | Why these numbers |
| assumptions | list[str] | Key assumptions made |
| confidence | float (0-1) | Confidence in this estimate |

### RiskAssessment
Output from Risk Agent.

| Field | Type | Description |
|-------|------|-------------|
| agent_name | str | "Risk Agent" |
| risk_factors | list[str] | Identified risk factors |
| risk_adjustment_pct | float | Suggested % price increase |
| reasoning | str | Why this adjustment |
| confidence | float (0-1) | Confidence in assessment |

### ValidationResult
Output from Validator Agent.

| Field | Type | Description |
|-------|------|-------------|
| is_valid | bool | Pass or fail |
| issues | list[str] | Problems found |
| suggestions | list[str] | Improvement suggestions |
| summary | str | One-paragraph assessment |

### PricingResult
Final output after full swarm run.

| Field | Type | Description |
|-------|------|-------------|
| price_floor | float | Final minimum viable price |
| price_target | float | Final recommended price |
| price_stretch | float | Final aspirational price |
| currency | Currency | Price currency |
| suggested_structure | str | Pricing format recommendation |
| rationale | str | How arbiter arrived at these numbers |
| assumptions | list[str] | All assumptions |
| risk_factors | list[str] | All risk factors |
| confidence | float (0-1) | Overall confidence |
| specialist_proposals | list[AgentProposal] | Individual agent outputs |
| risk_assessment | RiskAssessment | Risk agent output |
| validation | ValidationResult or None | Validator output |
